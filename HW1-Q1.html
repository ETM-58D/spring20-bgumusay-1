B For 2D, fraction valuse should be equal to the area of a circle having r=1 over the area of square having a=2. From this equation we can estimate pi value 2D. For 3d, fraction valuse should be equal to the volume of sphere having r=1 over the volume of a cube having a=2. From this equation we can estimate pi value for 3D. Both of those values are very close to the actual pi and the difference results from using limited number of observations, if we could use infinite number of observations then true values will be much more closer.

pi_estimationD2 <- (fractionD2*4)/1
print(paste("Esitmation for pi using 2D is",pi_estimationD2))
## [1] "Esitmation for pi using 2D is 3.128"
pi_estimationD3 <- (fractionD3*8)/(4/3)
print(paste("Esitmation for pi using 3D is",pi_estimationD3))
## [1] "Esitmation for pi using 3D is 3.114"
pi_actual <- pi
print(paste("Actual pi value is",pi_actual))
## [1] "Actual pi value is 3.14159265358979"
